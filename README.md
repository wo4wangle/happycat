## 💻 how to deploy
1. build jar with idea 
2. copy b/、cong/、javaweb/、lib/、log/、webapps/、work/  to the directory of jar file
3. config right docbase of happycat/conf/server.xml 
4. run with "java -jar happycat.jar"
